# UPI_FRAUD_DETECTION_AND_RISK_ANALYSIS_DASHBOARD
## Project Overview
I worked on this project to explore how fraud patterns appear in UPI transactions.
By analyzing the data, I identified trends like peak fraud hours, high-risk locations, and the banks most affected. One key observation was that fraud activity increases during late-night hours.
## Key Insights
- Fraud Rate: 11.2%
- Peak Fraud Time: 1 AM – 4 AM
- High-Risk Location: Hyderabad
- Highest Fraud Bank: Axis Bank
## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard Design
## Conclusion
Fraud transactions are more frequent during late-night hours, indicating a pattern in suspicious activity.
